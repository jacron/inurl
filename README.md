# inurl
show images on an index page

achtergrond: eerste versie van images tonen op index page was via tampermonkey

```==UserScript==
// @name         openindeximgs
// @namespace    http://tampermonkey.net/
// @version      0.11
// @description  on Index page, immediately load the images
// @author       Jan Croonen
// @match        *://*/* 
// @grant        none
// bookmarklets/opener
// ==/UserScript==
```
